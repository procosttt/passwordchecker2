# Проверка надежности пароля

Мини-сервис на Flask для оценки надежности пароля.

## Что умеет
- проверяет длину пароля;
- определяет наличие букв, цифр, спецсимволов и верхнего регистра;
- находит простые шаблоны вроде `1234`, `abcd`, `qwerty` и повторы;
- предупреждает о личных данных: имени и году рождения;
- выдает уровень надежности и рекомендации.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000
```
